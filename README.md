## Projeto 💻
Você pode testar a aplicação nos endereços abaixo:
<br />
 - Versão WEB: [https://www.bethehero.kennedyrs.tk](https://www.bethehero.kennedyrs.tk)
 - Aplicativo na Play Store: [Baixar](https://play.google.com/store/apps/details?id=tk.kennedyrs.bethehero&hl=pt_BR)

<p>A Semana Omnistack é um evento online e gratuito produzido pela empresa Rocketseat, com o objetivo de entregar uma imersão do aluno na stack Javascript.</p>
<p>Neste evento o projeto foi o Be The Hero, uma aplicação web e mobile que visa ajudar ongs a lançarem campanhas para arrecadar dinheiro para o tratamento de algum pet.</p>
<p>Após o anúncio ser encontrado, o possível doador poderá entrar em contato com a ong através do whatsapp ou e-mail.</p>
